Palavras-chave: Analista de sistemas, desenvolvimento de sistemas, Analista de negócio, Teste, Analista de teste QA, Sustentação, Git, Git Flow, front, frontend, java script, html, css.

Sou uma profissional com facilidade em aprender e se adaptar em equipes de grande performance, comunicativa e criativa com dinamismo e energia positiva para motivar e melhorar o ambiente de trabalho capazes de motivar equipes, inspirar colegas de trabalho, acompanhar as mudanças do mundo e do grupo e se relacionar bem socialmente. Com foco na entrega final (E2E), de documentação, sustentação e manutenção de sistemas para a organização. Forte atuação no Relacionamento interpessoais, trabalhando na elicitação e elucidação de requisitos de negócio para entrega de resultados eficazes, tempestivamente. Espírito empreendedor com grande aptidão para identificar problemas e desenvolver soluções para otimizar a rotina de trabalho.

Melhores práticas de gerenciamento de projetos tradicional e Agile, Scrum, Escrita de User Story, sistemas operacionais Windows, Tecnologias web HTML, JavaScript, CSS e Repositório Git.
